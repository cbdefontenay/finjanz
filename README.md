# Finjanz

### AusgabenTracker – Einfaches, lokales Ausgaben-Management

**AusgabenTracker** ist eine schlanke, datenschutzfreundliche Finanz-Übersichts-App für Android, die komplett **lokal** auf deinem Gerät arbeitet – ohne Cloud, ohne Tracking, ohne Registrierung.

**Hauptfunktionen:**
- Ausgaben schnell erfassen mit Betrag, Kategorie und Notiz  
- Beliebig viele eigene Kategorien anlegen  
- Übersichtliche Statistik: Monats-/Jahresansicht, Kategorien-Verteilung, Trends  
- Diagramme und Summen für bessere Budget-Kontrolle  
- Kompletter Export als CSV-Datei (für Excel, LibreOffice etc.)  
- 100 % offline – deine Daten verlassen nie dein Smartphone  

**Technik:**  
- Entwickelt mit **Flutter**  
- **Open Source** (quelloffen)  
- Nur für **Android** optimiert  

Perfekt für alle, die eine einfache, werbefreie und wirklich private Haushaltsbuch-Alternative suchen.
